## 使用
### 在线使用
- [在我的 github page 上使用](https://shhjtvp.github.io/offerings/RedOS/index.html)：要开加速器。

### 离线使用
#### 配置要求
- 最好使用基于[Chromium](https://www.chromium.org)的浏览器。
- 内存>512mb；cpu性能>Intel Core i3-2500H；需要有核心显卡或独立显卡。
#### 其他
- 不要用于非法用途。
